# 🏏 CrickBot-RAG

AI-powered Cricket Chatbot using:

- LangChain
- Gemini API
- ChromaDB
- HuggingFace Embeddings
- Tavily Search
- RAG Architecture

---

# 🚀 Features

✅ PDF-based Question Answering  
✅ Cricket Knowledge Retrieval  
✅ Gemini LLM Integration  
✅ Web Search Fallback  
✅ Chroma Vector Database  
✅ Interactive Chatbot  

---

# 🛠️ Tech Stack

- Python
- LangChain
- Google Gemini
- ChromaDB
- HuggingFace
- Tavily Search
- Google Colab

```env

```
